# Installation {#installation}

## Using pip

```bash
pip install rag-sdk
```

## Using poetry

```bash
poetry add rag-sdk
```

## With specific providers

To install with specific dependencies to keep your environment light:

```bash
pip install rag-sdk[openai,pinecone,anthropic]
```

## Development installation

For contributors or those who want to run the full test suite:

```bash
pip install rag-sdk[dev]
```
